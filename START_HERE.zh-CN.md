START HERE

（给营养师与非技术用户）

这个文件夹用于把
Mineral Balance Diet 安卓应用
中的食谱，制作成 带语音讲解的短视频。

你不需要任何编程知识。
只需要 按照下面的步骤，一步一步操作 即可。

这个工具能做什么？

它可以把：

• 手机上显示的食谱页面
• 你写的简单说明文字

变成：

• 一个带语音讲解的短视频文件

最终生成的视频文件名是：

output.mp4

这些设置只需要做一次

下面的设置步骤 只需要做一次。
完成后，每次制作视频都会非常简单、重复操作即可。

第一步 —— 安装 Python（只做一次）

Python 是一个用来运行本工具的辅助程序。
你 不需要学习编程。

操作步骤：

• 打开浏览器
• 访问 python.org
• 下载 Python 3
• 运行安装程序

重要提示：

在安装界面中，请一定勾选
“Add Python to PATH”

安装完成后，Python 就可以使用了。

第二步 —— 安装 ffmpeg（只做一次）

ffmpeg 用来生成视频文件。

操作步骤：

• 打开浏览器
• 搜索 “ffmpeg Windows 下载”
• 下载 Windows 版本

下载完成后：

• 打开下载的文件夹
• 找到这两个文件：
– ffmpeg.exe
– ffprobe.exe

• 把 这两个文件
• 复制到 当前这个文件夹
（也就是和 RecipeVideoMaker.py 在同一个文件夹）

完成。

第三步 —— 连接安卓手机（只做一次）

你需要允许电脑访问手机屏幕。

在安卓手机上：

• 打开 设置
• 进入 关于手机
• 连续点击 版本号 多次
• 直到开启开发者模式

然后：

• 返回设置
• 打开 开发者选项
• 打开 USB 调试

接着：

• 用 USB 线把手机连接到电脑
• 如果手机弹出提示，请点击 允许

第四步 —— 找到你的设备 ID（只做一次）

设备 ID 是电脑识别你手机的编号。
你 不需要理解它是什么，只需要复制一次即可。

如何找到设备 ID

请确认：

• 手机已经通过 USB 连接电脑
• 手机已开启 USB 调试

在电脑上：

• 打开 Windows 命令行窗口
（点击开始菜单，输入 cmd，按回车）

在命令行窗口中输入：

adb devices

然后按回车。

你会看到一行内容，类似下面这样
（你看到的字母和数字会不同）：

26111JEGR11210 device

👉 前面的 字母和数字，就是你的 设备 ID。

第五步 —— 把设备 ID 写入 device.txt（只做一次）

在这个文件夹中，有一个文件叫 device.txt。

操作方法：

• 打开 device.txt
• 把刚才看到的设备 ID 粘贴进去
• 确保 只占一行
• 保存文件

以后不需要再修改，
除非你换了手机。

设置完成后：每次制作视频的步骤

下面这些步骤，
是你 每次制作新视频时要重复的操作。

步骤 A —— 在应用中打开食谱

• 在手机上打开 Mineral Balance Diet 应用
• 打开你要制作视频的食谱
• 停留在 第一个食谱页面

步骤 B —— 逐页截图

在这个文件夹中，有一个文件叫 shot.bat。

这个文件 每运行一次，只截取一张当前屏幕。

工作方式：

• 截取手机当前显示的页面
• 把图片保存到当前文件夹
• 自动命名为：
01.png、02.png、03.png……

操作方法

• 手机显示第一个食谱页面
• 在命令行窗口中输入：

shot.bat 1

• 会生成 01.png

然后：

• 手机上滑动到下一个食谱页面
• 在命令行窗口中输入：

shot.bat 2

• 会生成 02.png

按这个方式继续，
直到所有需要的页面都截图完成。

重要规则：

• 一页 = 一张图片
• 必须按顺序操作
• 不要跳过数字

步骤 C —— 编写语音文字（script.txt）

在这个文件夹中，有一个文件叫 script.txt。

这个文件里的文字，就是视频中要播放的语音内容。

规则很简单：

• 一行文字 = 一张图片
• 第一行对应 01.png
• 第二行对应 02.png
• 第三行对应 03.png

建议：

• 每行一句话
• 内容简短、清楚

步骤 D —— 生成视频

在这个文件夹中，有一个文件叫 RecipeVideoMaker.py。

要生成视频，需要在 Windows 命令行窗口中运行它。

操作方法：

• 打开 Windows 命令行窗口
（开始菜单 → 输入 cmd → 回车）

• 确认命令行窗口所在位置
是当前这个项目文件夹
（包含 RecipeVideoMaker.py 的文件夹）

在命令行窗口中输入：

python RecipeVideoMaker.py

然后按回车。

程序会自动完成以下工作：

• 读取图片（01.png、02.png……）
• 读取 script.txt
• 自动生成语音
• 合成为一个视频文件

完成后，你会看到一个新文件：

output.mp4

这就是最终生成的视频。

重要提示

• 图片必须命名为 01.png、02.png 这样的格式
• 不要随意重命名文件
• 所有文件必须放在同一个文件夹中

一句话总结

截图 → 写文字 → 运行 RecipeVideoMaker → 得到视频
